# ⚡ ESP32 Arduino Projects

A small collection of **random ESP32 projects** built using the **Arduino IDE**.

This repository is basically a playground — different ideas, experiments, and quick prototypes.

---

## 🧪 What to expect

* Wi-Fi / Bluetooth experiments
* Sensors and simple automation
* LED control and small hardware hacks
* Occasional weird ideas that somehow work

Projects are mostly independent and may vary in quality and style.

---

## 🚀 Usage

1. Open any project in **Arduino IDE**
2. Select your ESP32 board
3. Upload and run

---

## ⚠️ Notes

* Code can be experimental
* Not everything is polished
* Some projects may require additional libraries
